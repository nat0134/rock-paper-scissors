# rock-paper-scissors

A basic Javascript project for The Odin Project creating a "Rock Paper Scissors" game between user and computer, winner and score is announced after full game is finalised. The game is played entirely in the console.

## Live Demo

https://nat0134.github.io/rock-paper-scissors/

## About The Project

This project demonstrates basic front-end web developement skiils using Javascipt. The program consists of function logics prompting user inputs to participate a game with the computer, Math.random() built-in function is used to generate computer player input. Scores are tracked and calculated to determine the winner of the full game.

It presents an understanding and use of:
- Basic functions
- if/else statements
- Comparison logic
- Loops
- Scope
- Nested functions
- Built-in functions
- Syntax
- Browser developer tools
